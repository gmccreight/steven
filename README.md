# Steven

'Even Steven'

Given a list of files and their sizes, make a naive attempt to split them
into buckets based on their size

## Installation

Add this line to your application's Gemfile:

    gem 'steven'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install steven
